# Crypto

Este proyecto fue desarrollado con el objetivo de aprender y poner en práctica los siguientes conceptos y tecnologías:

## Tecnologías Utilizadas
- **React**: Para la construcción de la interfaz de usuario.
- **TypeScript**: Para tipado estático y mayor robustez del código.
- **Redux**: Manejo del estado global de la aplicación.

## Conceptos y Prácticas Aplicadas
- Estructuración modular del código, manteniendo responsabilidades únicas en cada componente.
- Manejo de estado global con Redux y actualización eficiente de la UI.
- Consumo de APIs externas para obtener información de precios de criptomonedas en tiempo real.
- Validación y manejo de errores al consumir datos externos.
- Uso de componentes reutilizables y separación clara de lógica y presentación.
- Aplicación de buenas prácticas como nombres descriptivos, código legible y principio KISS (Keep It Simple, Stupid).

## Aprendizajes Clave
- Integración de varias tecnologías modernas en un flujo de trabajo real.
- Importancia del tipado y validación de datos.
- Diseño de componentes React fáciles de mantener y escalar en proyectos reales.
